# Currency Converter Project

This is a simple currency converter project built using JavaScript, HTML, and CSS. Users can convert between different currencies by entering the amount and selecting the desired currencies.

## Features

- Convert between different currencies
- Real-time exchange rates using an API
- User-friendly interface

## Technologies Used

- JavaScript
- HTML
- CSS

## How to Use

1. Clone the repository to your local machine
2. Open the index.html file in your browser
3. Enter the amount you want to convert
4. Select the currencies you want to convert between
5. Click the convert button to see the result
